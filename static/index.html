Pre-Call Assistant

Generate a pre-call brief using local files, Google API, or MCP

About — Pre-Call Assistant

Starting the server

From the project root, start the FastAPI server with uv:

uv run python main.py

The server starts on port 8000 with hot-reload enabled.

Frontend client

Open the app in your browser:

http://localhost:8000

Environment setup

Fill in your keys in .env:

ANTHROPIC_API_KEY=sk-ant-...
ELEVENLABS_API_KEY=...        # for /transcribe (STT)
ELEVENLABS_VOICE_ID=...       # free premade voice ID
GMAIL_MCP_URL=...             # MCP mode only
GDRIVE_MCP_URL=...            # MCP mode only

For API mode, run the one-time Google OAuth flow first:

uv run python scripts/get_token.py

curl — Local mode

Reads context from data/*.txt files — no external APIs needed.

curl -s -X POST http://localhost:8000/brief \
  -H "Content-Type: application/json" \
  -d '{"query": "prep me for my call with ClientA", "mode": "local"}' \
  | python3 -m json.tool

curl — API mode

Fetches live context from Gmail and Google Drive via OAuth. Requires token.json.

curl -s -X POST http://localhost:8000/brief \
  -H "Content-Type: application/json" \
  -d '{"query": "prep me for my call with ClientA", "mode": "api"}' \
  | python3 -m json.tool

curl — MCP mode

Claude fetches context autonomously via MCP servers. Requires MCP URLs in .env.

curl -s -X POST http://localhost:8000/brief \
  -H "Content-Type: application/json" \
  -d '{"query": "prep me for my call with ClientA", "mode": "mcp"}' \
  | python3 -m json.tool

curl — Benchmark (all modes)

Runs all configured modes and returns a side-by-side timing comparison.

curl -s -X POST http://localhost:8000/benchmark \
  -H "Content-Type: application/json" \
  -d '{"query": "prep me for my call with ClientA"}' \
  | python3 -m json.tool

curl — Transcribe (speech-to-text)

Upload an audio file and get back the transcribed text via ElevenLabs STT.

curl -s -X POST http://localhost:8000/transcribe \
  -F "audio=@/path/to/recording.mp3" \
  | python3 -m json.tool

curl — Speak (text-to-speech)

Convert any text to speech via ElevenLabs TTS and save as MP3.

curl -s -X POST http://localhost:8000/speak \
  -H "Content-Type: application/json" \
  -d '{"text": "Deal status: negotiation stage, one eighty K ARR."}' \
  --output brief.mp3 && open brief.mp3
